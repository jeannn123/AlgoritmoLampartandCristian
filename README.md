# Simulador de Lamport y Cristian
Aplicacion web con Flask que presenta dos simulaciones en una sola pagina:
- Algoritmo de la panaderia de Lamport con cinco procesos, tickets y orden de acceso.
- Algoritmo de Cristian con un servidor de tiempo, tres clientes y compensacion de latencia.
Abre `http://127.0.0.1:5000` en el navegador.
